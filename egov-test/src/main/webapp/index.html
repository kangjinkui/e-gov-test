법령-조례 매칭 시스템

eGovFrame 기반 법령-조례 매칭 시스템입니다.

API 엔드포인트

법령 관리

GET /api/laws - 법령 목록 조회
GET /api/law-amendments - 법령 개정 목록
GET /api/law-changes - 법령 변경사항
GET /api/law-snapshots - 법령 스냅샷

조례 관리

GET /api/ordinances - 조례 목록
GET /api/ordinance-articles - 조례 조항
GET /api/ordinance-law-mappings - 법령-조례 매핑

기타

GET /api/departments - 부서 목록
GET /api/reviews - 검토 목록

초기 계정

  • 관리자: admin / admin123 (ROLE_ADMIN)
  • 일반 사용자: user / user123 (ROLE_USER)
